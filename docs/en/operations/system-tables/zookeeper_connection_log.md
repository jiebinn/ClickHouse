## Description {#description}

The 'system.zookeeper_connection_log' table shows the history of ZooKeeper connections (including auxiliary ZooKeepers). Each row shows information about one event regarding connections.

<Note>
The table doesn't contain events for disconnections caused by server shutdown.
</Note>

## Columns {#columns}

<!--AUTOGENERATED_START-->
- `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) тАФ Hostname of the server which is connected to or disconnected from ZooKeeper.
- `type` ([Enum8('Connected' = 0, 'Disconnected' = 1)](/sql-reference/data-types/enum)) тАФ The type of the event. Possible values: Connected, Disconnected.
- `event_date` ([Date](/sql-reference/data-types/date)) тАФ Date of the entry.
- `event_time` ([DateTime](/sql-reference/data-types/datetime)) тАФ Time of the entry
- `event_time_microseconds` ([DateTime64(6)](/sql-reference/data-types/datetime64)) тАФ Time of the entry with microseconds precision.
- `name` ([String](/sql-reference/data-types/string)) тАФ ZooKeeper cluster's name.
- `host` ([String](/sql-reference/data-types/string)) тАФ The hostname/IP of the ZooKeeper node that ClickHouse connected to or disconnected from.
- `port` ([UInt16](/sql-reference/data-types/int-uint)) тАФ The port of the ZooKeeper node that ClickHouse connected to or disconnected from.
- `index` ([UInt8](/sql-reference/data-types/int-uint)) тАФ The index of the ZooKeeper node that ClickHouse connected to or disconnected from. The index is from ZooKeeper config.
- `client_id` ([Int64](/sql-reference/data-types/int-uint)) тАФ Session id of the connection.
- `keeper_api_version` ([UInt8](/sql-reference/data-types/int-uint)) тАФ Keeper API version.
- `enabled_feature_flags` ([Array(Enum16('FILTERED_LIST' = 0, 'MULTI_READ' = 1, 'CHECK_NOT_EXISTS' = 2, 'CREATE_IF_NOT_EXISTS' = 3, 'REMOVE_RECURSIVE' = 4, 'MULTI_WATCHES' = 5, 'CHECK_STAT' = 6, 'PERSISTENT_WATCHES' = 7, 'CREATE_WITH_STATS' = 8, 'TRY_REMOVE' = 9, 'LIST_WITH_STAT_AND_DATA' = 10, 'GET_CHILDREN_RECURSIVE' = 11))](/sql-reference/data-types/array)) тАФ Feature flags which are enabled. Only applicable to ClickHouse Keeper.
- `availability_zone` ([String](/sql-reference/data-types/string)) тАФ Availability zone
- `reason` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) тАФ Reason for the connection or disconnection.
<!--AUTOGENERATED_END-->

```sql
SELECT * FROM system.zookeeper_connection_log;
```
