<Tip>
  **SQL コンソール**

  SQL クライアント接続が必要な場合は、ClickHouse Cloud サービスに関連付けられた Web ベースの SQL コンソールを利用できます。詳細は、以下の **SQL コンソールに接続** を展開してご確認ください。
</Tip>

<Accordion title="SQL コンソールに接続">
  ClickHouse Cloud サービスの一覧から、サービスをクリックします。

  <Image img="/images/_snippets/cloud-connect-to-sql-console.png" alt="SQL コンソールに接続" size="lg" force border />

  SQL コンソールにリダイレクトされます。

  <Image img="/images/_snippets/createservice8.png" alt="SQL コンソール" size="lg" force border />
</Accordion>
